WSU eTextReader Help

Welcome to the Wayne State University Library eText Reader. This application is designed to provide full-text access to electronic texts and resources. What you are seeing for each page, when not using text overlays, is a digitized image of the page. The Wayne State Library system has digitized many of these resources, a great deal of them for the first time. One advantage to this interface is a faithful reproduction of the original layout of the text. This reader is particularly suited for texts with complex layouts and images. If you have any further questions about the eText Reader, please don't hesitate to contact us.

Below is a quick overview of how to use this reader:

Navigation Tools: First Page, Previous Page, Next Page, Last Page
"Jump to Section": Table of Contents or other structured links for the text
"Jump to Page" and total number of pages.
Full-text Search
Page Image Zoom: Zoom Out, Zoom In
Tools: Text Overlay, Speak Aloud, Zoom Loupe, Full-Screen, Information about the Text, Help Window
Font Resizing: Increase Font, Decrease Font
(displayed only with Text Overlays or Plain Text are activated)
Reading Modes: One Page, Two Page, Thumbnails, Plain Text / HTML
